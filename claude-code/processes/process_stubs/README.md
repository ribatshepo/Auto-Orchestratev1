# Process Stubs

Quick-reference stub documents for high-frequency processes. Each stub is a lightweight
companion to the full process file.

## Status Legend

| Status | Meaning |
|--------|---------|
| STUB | Incomplete placeholder; full definition in main process file |
| INTEGRATED-ADVISORY | Integrated into process injection map as advisory hooks |
| PARTIAL-INTEGRATED | Some processes integrated, others remain organizational-only |
| POST-PIPELINE | Referenced at pipeline completion as informational |
| COMPLETE | Full standalone process definition |
| DEPRECATED | Superseded; kept for historical reference |
